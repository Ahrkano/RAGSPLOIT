[SERVICE: SSH]
BANNER PATTERNS: "SSH-2.0-OpenSSH", "dropbear", "libssh"
RECOMMENDED ACTION: Enumeration / Credential Testing
PRIMARY MODULE: auxiliary/scanner/ssh/ssh_version
SECONDARY MODULE: auxiliary/scanner/ssh/ssh_login
SPECIFIC CVEs:
- If banner contains "libssh 0.6" to "0.8": auxiliary/scanner/ssh/libssh_auth_bypass
NOTES: SSH rarely has direct RCE. Prefer version detection, user enumeration, and credential attacks.

- If banner contains "ProFTPD mod_copy": exploit/unix/ftp/proftpd_modcopy_exec
